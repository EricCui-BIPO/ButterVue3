# Shared Components

## 概述

本目录包含项目中的通用 Vue 组件，可在多个模块中复用。所有组件都遵循统一的设计规范和开发标准。

## 组件列表

### 数据展示组件

#### I0Table
- **功能**: 高级表格组件，支持分页、排序、自定义格式化
- **文档**: `./I0Table/README.md`
- **用途**: 数据表格展示、CRUD操作界面
- **特性**:
  - 零配置开箱即用
  - 智能数据格式化
  - 内置分页系统
  - 灵活插槽系统

### 交互组件

#### ActionDropdown
- **功能**: 下拉菜单触发器组件
- **文档**: `./ActionDropdown/README.md`
- **用途**: 表格行操作、卡片操作菜单
- **特性**:
  - 默认三点图标触发器
  - 支持自定义触发器
  - 灵活的触发方式配置
  - 多种弹出位置支持

### 展示组件

#### LocationDisplay
- **功能**: 地理位置展示组件
- **文档**: `./LocationDisplay/README.md`
- **用途**: 用户资料、表格中的地理位置显示
- **特性**:
  - 自动显示国家旗帜
  - 地理位置名称展示
  - 内置错误处理
  - 灵活的显示配置
