# 产品概述

## 1. 产品定位

### 1.1 产品名称
全球人力资源服务平台,代号 I0

### 1.2 产品类型
一站式全球人力资源服务平台，集成 EOR（雇主记录服务）+ GPO（全球薪酬外包）+ Contractor（独立承包人）三大核心服务

### 1.3 目标用户
- **主要用户**：需要全球人力资源服务的企业客户的管理者、HR、员工、独立承包人
- **服务用户**：服务交付团队成员（service delivery/operater center），
- **最终受益者**：企业管理者、HR、员工、独立承包人、服务交付人员。

### 1.4 竞品
- **Deel**：全球领先的人力资源科技平台，提供EOR、合同工管理等服务
  - 优势：技术领先、全球覆盖广、资金实力强
  - 劣势：服务本地化程度有限、价格较高

- **Global Partner**：专注于亚太地区的人力资源服务提供商
  - 优势：本地化服务深入、价格相对合理
  - 劣势：全球覆盖范围有限、技术平台相对落后



## 2. 产品价值主张

### 2.1 核心价值
为企业提供全球范围内的人力资源管理一站式解决方案，简化跨国人力资源管理复杂性，降低合规风险，提升运营效率。

### 2.2 解决的核心问题
- **合规复杂性**：不同国家的劳动法律法规差异
- **运营效率**：跨国人力资源管理流程繁琐
- **成本控制**：建立海外法律实体成本高昂
- **服务透明度**：缺乏统一的服务进度跟踪和数据分析
- **服务质量**：不同服务供应商质量参差不齐
- **客户服务**：提供24/7的客户支持和服务，确保客户需求得到及时满足
- **数据合规性**：确保客户数据在全球范围内的合规存储和处理
- **数据安全**：采用严格的安全措施保护客户数据，符合全球数据隐私法规

## 3. 产品服务模式

### 3.1 EOR（雇主记录服务）
**服务场景**：企业需要在海外雇佣员工但不想建立当地法律实体

**核心功能**：
- 员工入职办理
    - 入职过程中，采集资料（每个国家资料不同）
    - offer/合同签署(可以电子签，可以人工签上传附件留档)
    - 签证办理
    - 入职通知
- 员工离职处理
- 薪资调整管理（PSC）
- 续签合同服务
- 续签签证服务
- 每月薪资计算与发放服务

**服务流程**：
1. 客户作为独立租户在平台为员工下单入职、离职、PSC、续签等服务
2. 服务人员接收并处理服务请求
3. 双方实时跟踪流程节点和完成情况
4. 提供全球视角的统计报告和数据可视化分析

### 3.2 GPO（全球薪酬外包）
**服务场景**：企业拥有自己的法律实体，但希望将薪酬管理外包给专业服务商

**核心功能**：
- 薪酬日历计划（每个服务年初，可以计划未来一年每月的服务节点）
- 客户 HR 在每个薪资周期到来时，薪资计算准备数据收集。BIPO SD 可以代理办理
- 服务方提供薪酬计算，客户审核 G2N report.
- 服务费用线下结算
- 薪资发放执行，员工可以收到工资单

**服务流程**：
1. 客户或SD提供年度薪酬日历计划
2. 系统对每个薪酬组（pay group）每个薪资周期（pay period）到来时，按时间节点发送提醒
3. 客户 HR 上传薪资计算所需数据, 包括员工异动（增减），考勤休假，可变项（如报销）等。这个流程 BIPO SD 可以代办。
4. BIPO计算薪酬并提供 G2N report 给客户审核
5. 客户确认后支付薪酬+服务费（这部分线下处理）
6. BIPO 执行薪资发放
7. 员工收到工资单

### 3.3 Contractor（独立承包人服务）
**服务场景**：企业需要短期招聘独立承包人完成特定项目

**核心功能**：
- 独立承包人信息采集
- 合同签署
- 劳务报酬发放
- 项目周期管理

## 4. 平台架构特点

### 4.1 多端平台架构
平台采用四端分离的架构设计，为不同角色提供专门的操作界面：

- **Client Portal（客户端）**：面向企业客户的HR管理人员，提供服务订购、员工管理、流程跟踪等功能
- **Service Portal（服务端）**：面向服务交付团队（Service Delivery），处理客户服务请求、流程执行、质量管控等
- **Admin Portal（管理端）**：面向平台管理员，提供系统配置、用户管理、数据分析、运营监控等功能
- **Talent Portal（人才端）**：面向员工和独立承包人，提供个人信息管理、请假、报销、工资单查看、服务申请等功能

### 4.2 多租户架构
- 每个客户作为独立租户
- 数据隔离和安全保障，数据库存在法律法规要求的各地，中心化服务器汇总授权访问

### 4.3 全球化支持
- 多国家法律法规适配
- 多币种支持
- 多语言界面
- 多时区处理

### 4.4 全球化人力资源管理能力
平台为HR提供全球视角的员工管理和服务能力：

- **全球员工视图**：HR可以在统一界面查看全球范围内所有员工的在职状态、服务办理进度和关键信息
- **数据合规存储**：员工敏感信息按照各国法律法规要求存储在指定地区的服务器，确保数据合规性
- **全球化审批流程**：支持跨时区、跨地区的审批流程，包括：
  - 员工请假申请的全球化审批
  - 费用报销的多级审批流程
  - 薪资调整和晋升的审批管理
- **实时状态跟踪**：提供员工服务请求的实时状态更新和进度跟踪
- **合规性监控**：自动监控各地区的合规要求变化，确保员工管理符合当地法规
