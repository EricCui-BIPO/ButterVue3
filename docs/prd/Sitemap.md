# Sitemap

## Client 端
> 面向企业客户的HR管理人员，提供服务订购、员工管理、流程跟踪等功能

- Basic
    - Home - 仪表板首页，显示全球员工概览、待处理事项、服务状态统计
    - Message - 消息中心，接收系统通知、服务进度更新、重要提醒
    - Calendar - 日历视图，显示薪酬日历、重要节点、审批截止日期
- HR
    - Talents - 全球员工、Contractor 管理，员工信息查看、入职/离职办理、合同管理
    - Times - 考勤管理，员工请假申请审批、考勤数据查看、休假政策管理
    - Expenses - 费用管理，员工报销审批、费用统计分析、报销政策设置
    - Payroll - 薪酬管理，薪资计算、工资单审核、薪酬调整(PSC)申请
- Service
    - Orders - 服务订单管理，EOR/GPO/Contractor服务下单、进度跟踪
    - Projects - 项目管理，独立承包人项目跟踪、项目周期管理
    - Payments - 支付管理，服务费用支付、薪酬发放确认、财务对账
    - Invoice - 发票管理，服务发票查看、费用明细、财务报表
- Advanced
    - Reports - 数据报告，全球员工统计、服务质量分析、成本分析报告
    - Compliance - 合规管理，各国法规更新、合规风险监控、合规检查清单
    - Insights - 数据洞察，HR数据分析、趋势预测、决策支持
- Settings
    - Users - 用户管理，HR用户权限设置、角色分配
    - Roles - 角色管理，权限角色定义、审批流程配置
    - Access - 访问控制，数据访问权限、安全设置
- Logs
    - Active Log - 操作日志，用户操作记录、系统活动跟踪
    - Email Log - 邮件日志，系统邮件发送记录、通知历史
    - Error Log - 错误日志，系统错误记录、异常处理跟踪
    - Data Log - 数据日志，数据变更记录、审计追踪

## Talent 端
> 面向员工和独立承包人，提供个人信息管理、请假、报销、工资单查看、服务申请等功能

- Basic
    - Home - 个人仪表板，显示个人信息概览、待办事项、最新通知
    - Message - 个人消息，接收HR通知、服务更新、重要提醒
- HR
    - Profiles - 个人档案管理，基本信息维护、证件管理、紧急联系人
    - Times - 个人考勤，请假申请、考勤记录查看、休假余额查询
    - Expenses - 个人报销，费用报销申请、报销记录查看、收据上传
    - Payslips - 工资单查看，月度工资单下载、薪酬明细、税务信息
- Service
    - Invoice - 个人发票，独立承包人劳务发票、收入证明
    - Work Visa - 签证服务，签证申请进度、续签提醒、签证文件管理
    - Contract - 合同管理，劳动合同查看、合同续签、合同变更申请

## Service 端
> 面向服务交付团队（Service Delivery），处理客户服务请求、流程执行、质量管控等

- Basic
    - Home - 服务仪表板，待处理服务请求、工作负载、服务质量指标
    - Message - 服务消息，客户沟通、内部协作、紧急通知
    - Calendar - 服务日历，服务节点提醒、薪酬周期、截止日期管理
- HR
    - Talents - 员工服务管理，处理入职/离职、PSC申请、合同续签
    - Times - 考勤服务，协助客户处理考勤数据、休假审批
    - Expenses - 费用服务，协助处理员工报销、费用审核
    - Payroll - 薪酬服务，薪资计算、G2N报告生成、薪资发放执行
- Service
    - Orders - 服务订单处理，接收并执行客户服务请求、状态更新
    - Projects - 项目服务管理，独立承包人项目跟踪、交付管理
    - Payments - 支付处理，薪酬发放执行、服务费用结算
    - Invoice - 发票处理，服务发票生成、费用核算
    - Calendar Schedule - 服务排程，薪酬日历管理、服务节点规划
- Advanced
    - Reports - 服务报告，服务质量分析、效率统计、客户满意度
    - Compliance - 合规执行，确保服务符合各国法规、合规检查
    - Insights - 服务洞察，服务数据分析、流程优化建议

## Admin 端
> 面向平台管理员，提供系统配置、用户管理、数据分析、运营监控等功能

- Basic
    - Home - 系统仪表板，系统状态概览、性能指标、系统活动
    - Message - 系统消息，通知、系统事件、异常预警

- Service
    - Clients - 客户管理，企业客户信息、租户配置、服务套餐管理
    - Projects - 项目管理，EOR, GPO, 独立承包人项目跟踪、交付管理
    - Reports - 各业务分析报告

- Business Setup
    - Entity - 法律实体管理，全球法律实体信息、EOR服务实体配置
    - Location - 地区管理，服务覆盖地区、本地化配置、法规适配
    - Insights - 国家法律法规维护

- Settings
    - Users - 系统用户管理，平台用户权限、角色分配、访问控制
    - Roles - 系统角色管理，权限角色定义、功能模块配置
    - Access - 系统访问控制，安全策略、数据权限、API访问管理

- Logs
    - Active Log - 系统活动日志，平台操作记录、用户行为分析
    - Email Log - 邮件系统日志，邮件发送记录、通知系统监控
    - Error Log - 系统错误日志，平台错误监控、异常处理跟踪
    - Data Log - 数据操作日志，数据变更审计、合规性追踪
